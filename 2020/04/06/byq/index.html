⚡变压器的运行原理和主要特性
发表于 2020-04-06| 更新于 2020-05-22
阅读量:

#变压器的空载运行

参数:

  • 一、二次侧线圈匝数:\(N_1,N_2\)
  • 一、二次侧电压、电流:\(u_1,u_2,i_1,i_2\)
  • 一次侧空载电流、励磁电流:\(i_0,i_m\)
  • 一、二次侧磁通、漏磁\(\phi ,\phi_{1\rho }\)

由于铁磁材料存在饱和现象，主磁通与建立它的电流之间的关系是非线性的；
但漏磁通由于主要沿非铁磁材料闭合，他与电流\(i_0\)保持线性关系。

基本方程式

\(e_1 = -N_1 \frac {d\phi } {dt} , e_2 = -N_1 \frac {d\phi } {dt} , e_{1\rho } = -N_{1\rho } \frac {d\phi_{1\rho } } {dt}\)

\(e_1+e_{1\rho } = i_0r_1-u_1\)
\(u_1 = -(e_1+e_{1\rho })+i_0r_1\)

写成复数形式:
\(\dot {U_1} = -(\dot {E_1}+\dot {E_{1\rho }})+\dot {I_0}r_1\)

电动势与变比

忽略一次侧电阻压降与漏磁通:\(u_1 = -e_1\)
如外施电压按正弦变化，设 \(\phi = \Phi_msin\omega t\)

\(\Phi_m\)——主磁通最大值
\(\omega\)——磁通角频率,\(\omega = 2\pi f\)

\(\therefore e_1 = -N_1\frac {d\phi} {dt} = -\omega N_1\Phi_m cos\omega t =E_{1m}sin(\omega t - 90^{\circ}) \)
\(\therefore E_1 = \frac {E_{1m}} {\sqrt {2}} = 4.44fN_1\Phi_m,\dot {E_1} = -j4.44fN_1\dot {\Phi}_m\)
同理 \(E_2 = \frac {E_{2m}} {\sqrt {2}} = 4.44fN_2\Phi_m,\dot {E_2} = -j4.44fN_2\dot {\Phi}_m\)

关于漏磁的计算

\(\psi_{1\rho } = N_1\phi_{1\rho } = L_{1\rho }i_0\)

由上可得，由于漏磁通不饱和，可以认为\(\phi_{1\rho }\)和\(i_0\)成正比，因此一次侧的漏磁电感为常数。

\(e_{1\rho} = -N_1\frac {d\phi_{1\rho}}{dt} = -L_{1\rho}\frac {di_0}{dt}\Rightarrow -\dot {E_{1\rho}} = -j\omega L_{1\rho}\dot {I}_0 = -jx_{1\rho}\dot{I}_0\)
\(\therefore \dot {U_1} = (-\dot {E_1})+j\dot{I}_0x_{1\rho}+\dot{I}_0r_1=(-\dot{E}_1)+\dot{I}_0Z_1\Rightarrow \dot{U}_1\approx (-\dot{E}_1)\)

#变压器的负载运行

追加参数

  • 一、二次侧磁动势、合成磁动势:\(\dot{F}_0 = \dot{I}_0N_1,\dot{F}_2 = \dot{I}_2N_2,\dot {F}_m = \dot {I}_1N_1+\dot {I}_2N_2\)
  • 一、二次侧漏磁通，感应漏磁电动势:\(-\dot{E}_{1\rho}=jx_{1\rho}\dot {I}_1,-\dot{E}_{2\rho}=jx_{2\rho}\dot{I}-2\)

\(\dot{U}_1 = (-\dot{E}_1)+\dot{I}_1Z_1,\dot{U}_2 = \dot{E}_2-\dot{I}_2Z_2\)

磁动势平衡方程

  • \(\dot{F}_1+\dot{F}_2 = \dot{F}_m\)

\(\therefore \dot{I}_1 = \dot{I}_m+(-\dot{I}_2\frac {N_2}{N_1}) = \dot {I}_m+\dot{I}_{1L}\)

其中\(I_{1L}\)为负载分量，当\(I_m\)可以忽略时，\(I_1\approx I_{1L}\)，有\(\frac {I_1}{I_2}\approx \frac {1}{k}\)

变压器运行时，二次侧的电压取决于一次侧的电压。输出电流、功率和功率因数取决于二次侧所接的负载，而一次侧的输入电流、功率和功率因数则取决于二次侧的对应量。
